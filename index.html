Set Up Real-Time Sync

To enable live sync, you need a free Firebase project. Follow these steps once:

Go to console.firebase.google.com → Create a project (disable Analytics is fine).
In your project: Build → Realtime Database → Create Database → choose any region → start in test mode.
Go to Project Settings ⚙ → General, scroll to "Your apps" → click </> Web → register app → copy the databaseURL (looks like https://your-project-default-rtdb.firebaseio.com).
Paste the Database URL below and click Connect. Share this same HTML file with everyone — they just open it and it syncs!
